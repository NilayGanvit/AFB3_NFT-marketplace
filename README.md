Project Description - 

This NFT MARKETPLACE project is being made under the IITI Summer of Code '22!
It is a web platform in which user will be able to buy, mint ,and do auctions of their NFTs (Non Fungible Token). It will be made on the Polygon testnet . For Smart Contract development enviroment Hardhat is used .Ethers library are used for interaction between the frontend and blockchain . Next.js and Tailwind CSS are used.
